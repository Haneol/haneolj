📌 개요

이 웹사이트는 로컬 Obsidian에서 작성한 마크다운 노트를 자동으로 웹사이트로 변환하여 보여주는 프로젝트입니다. GitHub 저장소(private)와 연동하여 특정 브랜치의 마크다운 파일이 업데이트될 때마다 웹사이트의 내용도 자동으로 업데이트됩니다.

✨ 주요 기능

자동 동기화

GitHub 웹훅을 통해 마크다운 파일이 업데이트되면 웹사이트 콘텐츠도 자동으로 업데이트됩니다.

그래프 뷰

마크다운 파일 간의 링크를 분석하여 그래프를 생성, 시각화합니다(옵시디언 그래프 뷰와 유사하게 구현).

카테고리 뷰

디렉토리 구조를 기반으로 문서를 카테고리별로 정리하여 보여줍니다.

모바일 반응형

어떠한 기기에서든 최적화된 사용자 경험을 제공하기 위해 반응형 레이아웃을 적용했습니다.

💻 사용한 기술 스택

백엔드
프론트엔드
데이터 처리 & 시각화
도구 & 인프라

🔄 워크플로우

  1. 로컬 Obsidian에서 마크다운 문서 작성 및 편집
  2. GitHub 저장소에서, 배포 브랜치에 변경사항을 Merge하거나, 또는 직접 Push
  3. GitHub 웹훅이 웹서버에 변경 알림 전송
  4. 웹서버가 최신 내용을 Pull하고 해당 부분 캐시 갱신
  5. 웹사이트에 변경사항이 자동으로 반영
